Sessions

Sessions provide the core organizational structure of CaimCore, connecting the various phases of analysis, and providing persistent storage.

CaimCore.Session — Type
Session(path[, storage])

A structure for storing session information for a CaimCore analysis.

source
CaimCore.SessionStorage — Type
SessionStorage([version])

Persistent storage component of a CaimCore Session. This data will be saved and read from disk via calls to save and load.

julia> SessionStorage()
SessionStorage(v"0.1.0")
source
CaimCore.save — Function
save(filename, session)

Save the persistent component of a Session to a disc.

source
